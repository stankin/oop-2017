Описание тега < div >

Элемент < div > является блочным элементом и предназначен для выделения фрагмента документа с целью изменения вида содержимого.
Как правило, вид блока управляется с помощью стилей. Чтобы не описывать каждый раз стиль внутри тега, можно выделить стиль во внешнюю таблицу стилей,
а для тега добавить атрибут class или id с именем селектора. Как и при использовании других блочных элементов, содержимое тега
всегда начинается с новой строки.
После него также добавляется перенос строки.

Синтаксис

< div >... < /div >

Атрибуты

align – Задает выравнивание содержимого тега <div>.
title – Добавляет всплывающую подсказку к содержимому.
Также для этого тега доступны универсальные атрибуты и события.
